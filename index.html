Cascading Styling Sheet

CSS is the language used to style and control the appearance of HTML elements on a web page.

It can change fonts, colors, layout and other more.

Selectors

  1. Element Selector: Targets all elements of a specific type.
  2. Class Selector: Targets elements with a specific class attribute.
  3. ID Selector: Targets a single element with a specific ID attribute.
  4. Attribute Selector: Targets elements based on their attributes.

Types of CSS

  • Inline CSS: Styles applied directly within an HTML element using the style attribute.
  • Internal CSS: Styles defined within a style tag in the head section of an HTML document.
  • External CSS: Styles defined in a separate .css file linked to the HTML document.
